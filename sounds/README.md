# Sounds Folder

## Required Files

### ringtone.mp3
**Purpose:** Klingelton für eingehende Notrufe

**Empfohlene Eigenschaften:**
- Format: MP3
- Länge: 3-5 Sekunden
- Lautstärke: Moderate (wird im Code auf 0.7 gesetzt)
- Typ: Klassischer Telefon-Klingelton oder Sirene

**Wo beziehen:**
- Kostenlose Sounds: https://freesound.org/
- Such-Begriffe: "phone ring", "emergency ring", "alert tone"

**Alternative (falls keine Datei vorhanden):**
Das System funktioniert auch ohne Sound-Datei, es wird dann nur ein 404-Fehler in der Konsole angezeigt, aber die Funktionalität ist nicht beeinträchtigt.

## Andere mögliche Sounds (optional)

- `notification.mp3` - Für System-Benachrichtigungen
- `alert.mp3` - Für wichtige Warnungen
- `status-change.mp3` - Für FMS-Statusänderungen
- `mission-complete.mp3` - Für abgeschlossene Einsätze

## Hinweis

Diese Sounds sind optional und dienen nur der Verbesserung der Benutzererfahrung. Der Simulator funktioniert vollständig auch ohne Audio-Dateien.
